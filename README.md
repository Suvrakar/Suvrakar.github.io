## Portfolio Architecture

```
project/
├── index.html              # Entry point
├── assets/
│   ├── data.js             # ← Edit this to update content
│   ├── main.js             # DOM renderer (reads from data.js)
│   ├── style.css           # Design system + all styles
│   └── images/
│       ├── profile.jpg
│       ├── logos/          # Institution & company logos (local)
│       └── *.jpg           # Award & research photos
└── README.md
```

**To update the portfolio**, edit only `assets/data.js` — no build step required.
